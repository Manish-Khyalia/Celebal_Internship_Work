# Week 1: Basic Data Exploration and Cleaning using Pandas

## Objective
Learn Python basics and perform basic data exploration and cleaning using Pandas.

## Tasks Performed
- Loaded CSV dataset into a Pandas DataFrame.
- Explored data using head(), tail(), shape, columns and dtypes.
- Identified and handled missing values.
- Selected important columns and filtered rows.
- Checked and removed duplicate rows.
- Created quantity and total_amount columns.
- Saved the cleaned dataset as cleaned_dataset.csv.

## Files
- Week1.ipynb
- cleaned_dataset.csv

Original dataset used in this project was obtained from Kaggle.
