# Week 2: SQL Database and Querying

## Objective

The objective of this assignment is to design and query an E-Commerce Sales Database using SQL. It covers database creation, table relationships, constraints, data insertion, and SQL queries for data retrieval, filtering, aggregation, joins, and transactions.

## Database Name

**celebal_week2**

## Files Included

* **Week2.sql** – Complete SQL script containing database creation, table creation, sample data insertion, and solutions for Q1–Q27.
* **CEI_SQL_Task2_Week2.pdf** – Assignment report containing query results, screenshots, and brief explanations.

## Tables Created

* Customers
* Products
* Orders
* Order_Items

## SQL Concepts Covered

* CREATE DATABASE and CREATE TABLE
* Primary Key and Foreign Key
* Constraints (NOT NULL, UNIQUE, CHECK, DEFAULT)
* Indexes
* INSERT Statements
* SELECT Queries
* WHERE Clause
* GROUP BY and HAVING
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN
* CASE Statement
* Transactions (COMMIT and ROLLBACK)

## Outcome

Successfully designed an E-Commerce Sales Database, inserted sample data, and solved all assignment queries using SQL while demonstrating database relationships, indexing, joins, aggregation, and transaction management.
